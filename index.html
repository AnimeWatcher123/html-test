Searching Algorithms: Linear Search

Linear Search is a Searching Algorithm which iterates over the given list of objects in a array. When it meets the requried condition it will break the iteration look and return the desired object!


Let's look at an example of a given array with 5 objects and linear searching through that array and finding the desired object.

Name: Amos
Age: 20
Index: 0
Name: Clover
Age: 24
Index: 1
Name: Simran
Age: 22
Index: 2
Name: Raj
Age: 23
Index: 3
Name: Aaron
Age: 23
Index: 4


So here we have an array with 5 objects and let's search for a desired object, let's say Raj!

The linear search will start from the beginning i.e. the first object "Amos" and check if it's equal to "Raj". If yes! then it'll return the object at the given index.


So the linar search will keep progressing until it reaches the object at index 3 where in the search condition for "Raj" is true. Once the condition is true it returns the object at index 3.
